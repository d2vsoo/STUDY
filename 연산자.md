● 연산자
1. 산술연산자
 1) 사칙 연산자
   a. 더하기 +
   b. 빼기 - 
   c. 곱하기 * 
   d. 나누기 / 
 2) 나머지 연산자 %
 3) 증감 연산자
   a. 증가 ++
   b. 감소 --

2. 할당 연산자
 1) 변수에 값을 할당하는 연산자
 2) 사칙 연산자와 조합하여 사용 가능
 3) 할당 연산자 응용
   a. +=
   b. -=
   c. *=
   d. /=
   e. %=

3. 연결 연산자
 1) 문자열과 문자열을 연결
 2) + 기호 사용

4. 비교 연산자
두 값을 비교하여 true나 false로 결과값을 내는 연산자
 1) ==, !=
   두 개의 값이 같은지, 같지 않은지 확인
 2) <, <=
 3) >, >=
 4) ===. !===
   두 개의 값이 자료형까지 완벽하게 같은지, 같지 않은지 확인

5. 형변환
 1) 숫자형 + 문자형 : 숫자를 문자열로 인식
 2) 곱하기, 나누기, 나머지 연산에서는 문자형 자료를 모두 숫자로 자동 인식
